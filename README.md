# Veritas Deployment

**Veritas** is an infrastructure verification system that enforces **“verify every space before advancing”** – a doctrine for immutable, lineage‑preserving state vectors across data centers, water SCADA, civil engineering, and more.

This repository contains:

- SOC 2 readiness package (policies, evidence templates)
- Business Associate Agreement (BAA) template for healthcare pilots
- Minimal edge binary specification (Go, offline‑first)
- Failure Mode Matrix (FMM) stress test suite
- Deployment runbooks for DLR‑VA‑01 and water SCADA rigs
- Productization assets (SLA templates, pricing models, onboarding playbooks)

## Quick Start

See the [runbooks](./runbooks) for site‑specific instructions.  
For compliance, start with [SOC 2 readiness](./docs/soc2_readiness.md) and the [BAA template](./docs/BAA_template.md).

1. Verify every space before advancing  
2. 20 roads, one destination  
3. My system never lies to me

# Incident Response Policy

## Severity Levels
- **P0** – Ledger breach / total system failure → halt operations, notify within 1 hour
- **P1** – Quarantine events >1% of telemetry → investigate within 4 hours
- **P2** – Single guard violation → log and review weekly

## Response Playbook
1. Identify
2. Contain
3. Eradicate
4. Recover
5. Post‑mortem

## Forensic Evidence
Full ledger export required for any P0 incident.
